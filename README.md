# Checkout exercise

### Usage
```bash
sbt run
```

Then pass a list of items (apple or orange) separated with commas.
For example:
```bash
Enter items separated by ",": apple, orange, orange, apple, orange
Apple       :  60.0
Apple 2 x 1 :  0.0
Orange      :  25.0
Orange      :  25.0
Orange 3 x 2:  0.0
-------------------------------------
total        : 110.0
```
